# Background

The tinyspec skill prompts (`/tinyspec:refine`, `/tinyspec:work`, `/tinyspec:task`) sometimes need to ask the user questions (e.g., clarifying requirements, choosing between repos, handling blocked tasks). Currently they ask questions as inline text, but Claude Code provides an `AskUserQuestion` tool that presents structured, selectable options — a better UX.

# Proposal

Update all three skill prompts to instruct Claude to use the `AskUserQuestion` tool when asking the user questions, rather than asking inline.
