# Background

Presently the skills are prefixed by `spec-` (e.g., `/spec-refine`). Renaming them to use a `tinyspec:` prefix (e.g., `/tinyspec:refine`) better identifies them as belonging to this tool.

# Proposal

- Rename all skill slash commands and Rust constants from `spec-*` / `SPEC_*` to `tinyspec:*` / `TINYSPEC_*`.
